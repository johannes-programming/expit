Introduction
------------

This project provides an implementation of the expit function.

Installation
------------

To install expit, you can use pip. Open your terminal and run:

.. code-block:: shell

    pip install expit

Features
--------

``expit.function(x: float) -> float``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function implements expit.

``expit.main``
~~~~~~~~~~~~~~

This click command implements expit.

``expit.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its test function.

.. code-block:: python

    import expit
    expit.test()

Example
-------

To calculate the function value of 0.42 in the shell:

.. code-block:: shell

    py -m expit 0.42

Or inside of a Python script:

.. code-block:: python

    import expit
    print(expit.function(0.42)) #0.6034832498647263
